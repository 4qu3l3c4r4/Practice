# Scripts

## verify-dom-structure.js

Browser console script for verifying DOM selectors used in tests.

**Usage:**
1. Open your application in a browser
2. Open Developer Tools (F12)
3. Go to Console tab
4. Copy and paste the contents of `verify-dom-structure.js`
5. Press Enter to run

The script will check for common login form elements and list any data-testid attributes found on the page.
