# AI Context — Katalon Groovy

> This file is designed to be read by AI assistants (Kiro, Cursor, Copilot, Claude, ChatGPT, etc.) to help implement tests using this template.

## Template purpose

Script-mode testing using Katalon Studio with Groovy.

## Tech stack

Katalon Studio, Groovy/Java

## Discovery & test generation workflow

1. Run `scripts/discovery/discover-web-ui.js` against the target URL to extract all testable elements
2. Review the discovery report for selectors, forms, buttons, and navigation
3. Create test cases following the patterns in this template's README
4. See the template README for detailed setup and usage instructions

## Rules

- Follow existing patterns in the template
- Keep test data in `.env` or config files
- See README.md for full documentation
