# Scripts

## verify-dom-structure.js
Paste into browser DevTools Console to verify login selectors and list data-testid attributes.

## verify-selectors.ts
Run with `npx ts-node scripts/verify-selectors.ts` to verify selectors against a running app.

## test-reporting.js
Shows available test commands and report locations.

## generate-pdf-report.js
Generates a summary from test results JSON.
