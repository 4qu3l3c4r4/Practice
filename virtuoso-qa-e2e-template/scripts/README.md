# Scripts

## verify-dom-structure.js

Browser console script for verifying DOM elements and test attributes.

**Usage:**
1. Open your application in browser
2. Open Developer Tools (F12)
3. Go to Console tab
4. Copy and paste the script
5. Press Enter to run

**What it checks:**
- Login form elements (username, password, submit button)
- Lists all data-testid and data-test-id attributes
- Helps identify selectors for test automation
